Rozwiązanie zostało wykonane w Springu w Javie w wersji 11. Użyto IDE Intellij. W Intellij należy stworzyć nowy projekt wybierając "File" -> "New" -> "Project From Existing Sources...", należy wybrać podany folder ("currencyConverter") i w kolejnym okienku wybrać Maven w ramach opcji "Import project from external model". Wszystkie pliki projektu powinny się załadować i można uruchomić aplikację.
