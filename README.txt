Rozwiązanie zostało wykonane w Springu w Javie w wersji 11. Użyto IDE Intellij. W Intellij należy stworzyć nowy projekt wybierając "File" -> "New" -> "Project From Existing Sources...", należy wybrać podany folder ("currencyConverter") i w kolejnym okienku wybrać Maven w ramach opcji "Import project from external model". Wszystkie pliki projektu powinny się załadować i można uruchomić aplikację klikając w przycisk Run (zielony trójkąt na pasku do góry po prawej, w konfiguracji budowania aplikacji plik CurrencyConverterApplication powinien być zaznaczony jako plik z funkcją main rozpoczynającą działanie aplikacji). Główny kod aplikacji znajduje się w src/main/java/com.example.currencyConverter/.
